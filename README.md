# Quick Poll
これはDiscord向けの投票BOTです。「[Simple Poll](https://top.gg/bot/simplepoll)」のコマンドと互換性があります。  
ただし、任意の絵文字を選択肢として使用できたり、投票数の集計機能を持っています。
